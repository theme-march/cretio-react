React Installation

Please follow the instructions to see how you can install the project on your local machine:

  1. For local host -
    1. Open your terminal / command prompt in the project root
    2. Run: npm install
    3. Run: npm run dev (to start the dev server at http://localhost:5173)


  1. For Production Build -
    1. Open your terminal
    2. Run: npm run build (this will generate the dist folder in the root directory)
  2. Build and Deploying to Vercel - https://www.youtube.com/watch?v=7T4w0QJtL-o

Basic Site Settings

Change Site Title and Favicon

To change your Site title Favicon and Page Titles; open the cretio-react-documentation in your editor and go to the location by following screenshot which are given bellow:

  1. Update Site Title

    Navigate to the root folder and open index.htmlOpen the file . Locate the <title> tag inside the <head> section and update the text to your desired site name.

  2. Manage Favicon

    Replace the default icon file at srcassetsimglogofavicon.svgReplace the file with your custom icon.

Change Logo

To update the logos across the entire site, change the following files:

  • Header & Sidebar: srclayoutheaderheader.tsx & OffCanvasSidebar.tsx
  • Footer: srclayoutfooterfooter.tsx

Change Page Title

To change the main heading (title) of an individual page, go to srcpages and open the file for the page you want to edit (e.g., About.tsx, Services.tsx). Then locate the Breadcrumb component and change the title and highlightWords (optional) to your desired text.

Customize Menu

The navigation menu is dynamically generated from a JSON configuration file. To manage your menu items, follow these steps:

  1. Navigate to: srcdataJsonnavItemsList.jsonOpen the file

  2. Then customize the menu

Customize Footer

To customize the footer data, follow these steps:

  1. From the project folder, go to src layout footer footer.tsx
  2. Then customize the footer settings (Social links, Contact info, etc.) directly in the constants at the top of the file.

Customize Blog Data

To manage your blog posts, follow these steps:

  1. From the project folder, go to src dataJson blogsData.json Open the file
  2. Then customize the blog data or append new objects to the array (ensure each has a unique id).

Customize Services List Data

To manage your services data (including section headers), follow these steps:

  1. From the project folder, go to src dataJson servicesData.json Open the file
  2. Then customize the sectionInfo and mainServices arrays to update your content.

Customize Team List Data

To manage your team members across different sections, follow these steps:

  1. From the project folder, go to src dataJson teamMembersData.json Open the file
  2. Then customize the member profiles in the standardTeam or marketingTeam arrays.

Customize Portfolio Data

To manage your projects and categories, follow these steps:

  1. From the project folder, go to src dataJson projectsData.json Open the file
  2. Then customize the categories and mainProjects arrays. Ensure project tags match the filter selectors defined in categories.

Customize Error Pages

To update the error messages or visual elements, follow these steps:

  1. From the project folder, go to src pages ErrorPages 404.tsx
  2. Then customize the error headline, description, and "Back to Home" button text directly in the JSX.

Customize Header

The header is a dynamic component. You can modify the header and sidebar markup there.

  1. From the project folder go to src layout header header.tsx Open the menu you want to use
  2. Then customize the header and navigation data
